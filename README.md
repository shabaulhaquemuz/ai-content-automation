# 🤖 AI Content Automation Pipeline

A fully automated daily content system built with Make.com and free APIs.

## 🔄 How It Works
Idea → Script → Voice → Video → YouTube (runs daily at 7:30 AM automatically)

## 🛠️ Tools Used
| Tool | Purpose |
|------|---------|
| Make.com | Automation orchestration |
| Groq (LLaMA) | AI script generation |
| VoiceRSS API | Text to speech |
| Google Drive | Audio file storage |
| Shotstack API | Video creation |
| YouTube Data API | Auto upload |

## 📋 Pipeline Steps
1. Groq AI generates a daily script about AI topics
2. VoiceRSS converts script to MP3 audio
3. Audio saved to Google Drive
4. Shotstack creates MP4 video with background + title + voice
5. Video auto-uploaded to YouTube via YouTube Data API

## 💰 Total Cost
$0 — Built entirely on free tiers
